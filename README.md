# Inventory Management Dashboard (Streamlit + MySQL)

This is an Inventory Management and Supply Chain Dashboard built using:
- Streamlit (Frontend UI)
- MySQL (Database)
- Python (Backend)
- Pandas/Numpy

## Features
- Basic Metrics (Total Suppliers, Total Products, Categories)
- Sales & Restock Values (Last 3 Months)
- Supplier & Product Details
- Reorder Alerts

## How to Run
1. Install dependencies:
    ```
    pip install -r requirements.txt
    ```

2. Create your MySQL database.

3. Add your database credentials inside:
    ```
    config/db_config.py
    ```

4. Run the app:
    ```
    streamlit run app.py
    ```
